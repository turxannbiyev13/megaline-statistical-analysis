# Megaline Statistical Analysis

## Project Overview

This project analyzes customer behavior and revenue patterns for **Megaline**, a telecommunications company. The objective is to compare the company's two prepaid plans (**Surf** and **Ultimate**) and determine whether there is a statistically significant difference in customer revenue.

The project includes data cleaning, exploratory data analysis (EDA), visualization, and hypothesis testing using Python.

---

## Objectives

- Analyze customer usage patterns
- Compare Surf and Ultimate plans
- Calculate monthly customer revenue
- Perform statistical hypothesis testing
- Visualize customer behavior and revenue distributions
- Provide data-driven business insights

---

## Dataset

The analysis uses five datasets:

- `megaline_calls.csv`
- `megaline_internet.csv`
- `megaline_messages.csv`
- `megaline_plans.csv`
- `megaline_users.csv`

## Statistical Methods

The project applies hypothesis testing to answer business questions, including:

- Is there a significant revenue difference between Surf and Ultimate plans?
- Does customer location affect average revenue?

Statistical significance is evaluated using **independent sample t-tests**.
